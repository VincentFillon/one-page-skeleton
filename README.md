# One Page Skeleton

Site vitrine moderne, maintenable et facilement personnalisable, basé sur Vite (template vanilla).

## Fonctionnalités
- Structure one page avec sections : À propos, Services, Contact
- Menu sticky en haut de la page pour une navigation rapide
- Système de thèmes (clair/sombre) via variables CSS et sélecteur
- Scroll fluide avec effet de snap entre les sections
- Mise en page responsive et professionnelle

## Installation

```bash
npm install
```

## Lancement du serveur de développement

```bash
npm run dev
```

## Personnalisation
- Modifiez le contenu des sections dans `index.html`
- Adaptez le style et les couleurs dans `src/style.css` (variables CSS)
- Ajoutez vos images, textes, coordonnées, etc.

## Déploiement

Pour générer la version production :

```bash
npm run build
```

Le dossier `dist/` contiendra les fichiers à déployer sur votre hébergement.
